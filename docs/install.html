Install (Chrome / Edge)

  1. Download sim-companies-extension-chrome.zip from the latest GitHub Release.
  2. Unzip it somewhere permanent (don’t delete the folder later).
  3. Open chrome://extensions.
  4. Enable Developer mode (top right).
  5. Click Load unpacked and select the unzipped folder (the one containing manifest.json).
  6. Go to Sim Companies → Retail building and you should see the “Retail Helper” panel.
