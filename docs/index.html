Sim Companies Retail Helper

A lightweight browser extension for simcompanies.com that shows retail profit/min, market cheapest listing, and your cost basis directly in-game.

Download latest release →

What it does

  • Displays real-time profit per minute for retail products
  • Shows the cheapest market listing
  • Calculates your cost basis automatically

Install (Chrome / Edge)

  1. Download sim-companies-extension-chrome.zip from the latest GitHub release .
  2. Unzip it somewhere permanent (don’t delete the folder later).
  3. Open chrome://extensions.
  4. Enable Developer mode (top right).
  5. Click Load unpacked and select the unzipped folder (the one containing manifest.json).
  6. Open Sim Companies → Retail building and you should see the Retail Helper panel.

Source & scope

This extension runs only on simcompanies.com pages. Source code and releases are hosted on GitHub.
